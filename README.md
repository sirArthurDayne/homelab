# Krosis Homelab

Repo dedicated to show all my Homelab activities that dont follow my other repos [Ansible Automations](https://github.com/sirArthurDayne/ansible-automations) & [Terraform Automations](https://github.com/sirArthurDayne/tf-automations).
## Objectives

Learning area for Kubernetes, DevOps and GitOps. 

## Paths
- ./apps: Used to test apply and delete apps manually with kubectl.
- ./clusters: Used for GitOps deployments with Fleet.
- ./utils: Miscelaneous scripts.

## Services

- K3s: easy and performance kubernetes.
- Kube-vip: Virtual IP Maganement.
- Metal-lb: Load Balancer and ip-address pool management.
- Rancher: Web GUI for managing the cluster.
- Fleet: for GitOps deployments.
- Linkding: personal bookmarks for reference.
- Nginx: Load balancer, mostly for "hello world" service and test.
- GrafanaStack : Monitoring the cluster.
- Longhorn: Distributed Storage for the cluster.

## Future Plans
